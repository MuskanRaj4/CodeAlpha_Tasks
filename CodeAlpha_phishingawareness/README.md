# Phishing Awareness Training

## Overview
This project is created to spread awareness about phishing attacks and cybersecurity threats.

## Topics Covered
- What is phishing
- Recognizing phishing emails
- Fake websites
- Social engineering attacks
- Safety tips and best practices
- Real-world examples
- Interactive quiz

## Objective
To educate users about online scams and how to stay safe from phishing attacks.

## Tools Used
- PowerPoint / Canva
- GitHub

## Key Learnings
- Identifying suspicious emails
- Detecting fake websites
- Protecting personal information
- Understanding cyber attack tactics

## Conclusion
Phishing attacks are increasing daily. Awareness and safe online practices help users avoid becoming victims.
